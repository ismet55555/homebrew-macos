# homebrew-macos

Homebrew tap for projects

## Install Tap and Install Formula

```bash
brew install ismet55555/things/<FORMULA NAME>
```

## Only Install Tap

```bash
brew tap ismet55555/things
```

## File Checksum

```bash
curl -L --output temp.tar.gz <GITHUB BIN URL>
shasum -a 256 out.tar.gz
rm temp.tar.gz
```
